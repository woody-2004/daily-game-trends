# Game Concepts — 2026-06-24

## Top Concept: "Settlement of Spires" — Cozy Roguelike Deckbuilder with Audience-Voted Runs

**Pitch:** A cozy roguelike deckbuilder (Slay the Spire 2 / Islantiles lineage) where each run also grows a small settlement, and — natively, not as a bolted-on stream overlay — a "town hall vote" mechanic lets a linked audience (TikTok/Discord) vote on key run decisions (which card to take, which building to place, fight vs. flee).

**Why now:**
- Rides the #1 confidence mechanic (roguelike deckbuilding) and #2 (cozy/systems hybrids).
- Fills market gap #1 (cozy + strategy, all-ages) and #3 (purpose-built TikTok-native interactive mechanic) simultaneously.
- Differentiates from Islantiles/Slay the Spire 2 by making the audience-vote loop a first-class feature rather than a streaming add-on — something none of the tracked competitors' covered titles do natively.

**Core loop:** Run a deck-based roguelike dungeon crawl → earn resources → spend resources expanding a cozy settlement between runs → optionally open "vote mode," where viewers pick the next card draft or building placement in real time.

**Risk:** Audience-vote mechanic is unproven as a core (not gimmick) feature — needs a single-player mode that's fully satisfying without an audience, per mechanics_confidence.md's "Medium-High, unproven as standalone product" note.

## Secondary Concepts

1. **"Recess League"** — a roguelike spin on a niche competitive genre (e.g., racing) with run-based deckbuilding for upgrades instead of grindy unlock trees, targeting gap #4 (mid-budget niche genres avoiding direct franchise competition).
2. **"Quietkeep"** — a wellness/cozy life-sim explicitly designed around short daily sessions (5–10 min), targeting gap #5 (gamified health) and gap #2 (non-core demographics), built by people with game-mechanic design experience rather than a health-app team.
