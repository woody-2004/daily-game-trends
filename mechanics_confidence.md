# Mechanics Confidence Tracker

**Methodology:** Confidence (0-10) reflects how strongly a mechanic/hook is currently validated by real-world evidence (concurrent players, revenue, cross-platform corroboration, structural market data). It is NOT the same as "Runs Seen" in `trends_summary.md` — a mechanic can have high day-1 confidence if the evidence is strong (e.g., an all-time CCU record), even before it has recurred across multiple runs.

Each run: if a mechanic reappears with new supporting evidence, increment **Days Seen** by 1 and adjust confidence up (strong new evidence) or hold flat (just persisting). If a mechanic stops appearing in discovery for several runs, mark direction "falling" and decay confidence slightly. Direction is "rising" / "flat" / "falling" based on trajectory since last run.
