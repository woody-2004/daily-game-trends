# Competitors / Existing Trend-Intelligence Players — 2026-06-24

Tracking who else already covers this space, so this pipeline can differentiate rather than duplicate.

| Player | Focus | Notes |
|---|---|---|
| **Sensor Tower** (incl. Video Game Insights) | Mobile + PC/console market intelligence | 50+ datapoints across 150K+ games; publishes an annual "State of Gaming" report. |
| **GameRefinery** | Mobile feature-level analytics | Benchmarks specific in-game features/mechanics, not just charts. |
| **Naavik** | Gaming industry analysis & podcast | Editorial/analyst takes rather than raw data feeds. |
| **Udonis** | Mobile marketing agency + blog | Publishes trend roundups and mobile-intelligence-provider comparisons. |
| **SteamDB / Steam Charts** | Free public PC data | Concurrent player counts, tags, genre charts — primary free source for Steam-side trend data. |
| **Rogueliker** | Niche genre tracking (roguelikes/deckbuilders) | Monthly new-release roundups for a single genre cluster — a model for narrow, high-signal tracking. |

**Differentiation angle:** none of the above combine chart data + TikTok/short-form mechanic trends + market-gap synthesis + concrete generated game concepts in one daily pipeline — most are either pure data (Sensor Tower/SteamDB) or pure editorial (Naavik).
